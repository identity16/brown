# Example Skill

This skill demonstrates the structure of a plugin-provided skill.

## When to use

Invoke this skill when the user explicitly asks for a brown plugin demo, or
when you need a reference for how skills bundled inside a plugin are
organized.

## How it works

1. Read the user's request to determine what they want to demo.
2. Respond with a short explanation of the skill structure.
3. Point the user at `skills/example-skill/SKILL.md` as the reference file.

## Notes

- Replace this file with the real skill instructions for your plugin.
- Additional resources (scripts, templates, references) can live alongside
  `SKILL.md` inside this directory.
